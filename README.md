# Laravel stater kit

## How to use

    - Click on Use this template button
    - follow normal project creation steps in git
    - Once project is created
    - cp .env.example .env
    - composer install
    - npm install && npm run dev
    - php artisan key:generate
    - php artisan migrate:fresh --seed
